<h2>Customer Demand Prediction and Client Segmentation</h2>

<p>
Система прогнозирования покупательской активности и сегментации клиентов для интернет-магазина.
</p>

<h3>Задача</h3>

<p>
Снизить отток клиентов и повысить эффективность маркетинговых кампаний за счёт персонализации предложений и раннего выявления пользователей со снижающейся активностью.
</p>

<h3>Решение</h3>

<p>
Разработал модель машинного обучения для прогнозирования снижения покупательской активности на основе поведенческих, маркетинговых и финансовых данных клиентов.
</p>

<p>
Объединил данные из нескольких источников, выполнил очистку аномалий, обработал неявные дубликаты и сформировал единый аналитический датасет.
</p>

<p>
Настроил пайплайн предобработки с кодированием категориальных признаков, масштабированием числовых данных и подбором гиперпараметров с использованием стратифицированной кросс-валидации.
</p>

<p>
Провёл интерпретацию модели с помощью SHAP и выполнил сегментацию клиентов по уровню прибыли и риску снижения активности.
</p>

<h3>Результат</h3>

<ul>
  <li>Достиг значения <strong>F1-score = 0.874</strong> на тестовой выборке.</li>
  <li>Определил клиентов с высоким риском снижения покупательской активности.</li>
  <li>Выявил ключевые факторы удержания: количество просмотренных страниц, число изученных категорий, время на сайте и история покупок.</li>
  <li>Сегментировал клиентскую базу по уровню прибыли и вероятности снижения активности.</li>
  <li>Выделил наиболее ценный сегмент клиентов с высокой прибылью и повышенным риском оттока.</li>
  <li>Сформировал рекомендации по персонализации маркетинговых кампаний и повышению удержания клиентов.</li>
</ul>
